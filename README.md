# Wyświetlacz planów zajęć — GitHub Pages

Ten projekt umożliwia automatyczne wyświetlanie planów zajęć (plików PDF) w interwale czasowym — np. na ekranach Smart TV lub w przeglądarce internetowej.

## Instrukcja użycia

1. Utwórz nowe repozytorium na GitHubie (np. `plan-zajec`).
2. Wgraj pliki `rotating-pdf-viewer.html` i `README.md`.
3. W GitHub → Settings → Pages wybierz:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Otwórz stronę pod adresem `https://twoja-nazwa.github.io/plan-zajec/`.
5. Strona automatycznie przełącza PDF-y co 20 sekund (można zmienić w ustawieniach).
